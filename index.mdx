<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with Sher in three simple steps</p>

  <Steps>
    <Step title="Install Sher globally">
      Install the Sher CLI using your preferred package manager:

      <CodeGroup>
      ```bash npm
      npm install -g shersh
      ```

      ```bash yarn
      yarn global add shersh
      ```

      ```bash pnpm
      pnpm add -g shersh
      ```

      ```bash bun
      bun add -g shersh
      ```
      </CodeGroup>
    </Step>

    <Step title="Navigate to your project">
      Open your terminal and navigate to any frontend project directory:

      ```bash
      cd my-vite-app
      ```

      Sher automatically detects your framework (Vite, Next.js, Astro, Create React App) and package manager.
    </Step>

    <Step title="Share your project">
      Run the link command to build and deploy:

      ```bash
      sher link
      ```

      Sher will build your project, upload it, and copy a live preview URL to your clipboard. The link expires in 24 hours by default (configurable with `--ttl`).

      <Tip>Run `sher login` to unlock higher limits: 25 links per day and 50MB uploads for free with GitHub authentication.</Tip>
    </Step>
  </Steps>
</div>
